Como Instalar?
==============

Antes de mais nada o Infra nao precisar ser instalado. Apesar disso, se você quiser testar a parte de persistência você precisará de outras duas bibliotecas free de terceitos:

Zeos - Este está compactado no diretório Third.
Dunit - Pode ser baixado da sourceforge.net (brevemente estarei disponibilizando compatado no third)

Sendo assim, para poder usar a persistência faça:

1) Descompacte o Arquivo Infra\Third\Zeos.Rar no próprio diretório, e dê um build All;
2) Baixe, compile e ponha no Library do Delphi a pasta com as units e dfms do Dunit;

Agora o Infra:

1) Baixe o ultimo release ou dê checkout dos fontes do repositorio
2) Jogue no diretório chamado Infra, ex: C:\Infra
3) No Delphi clique em Tools - Enviroment - Library
4) Adicione C:\Infra\Packages\D7\Build ao Search Path
5) Abra c:\Infra\Infra.bpg
6) Dê um Build All nos projetos e pode rodar os Testes.

Lista de diretórios e sua descrição:
====================================

Infra\Packages		 	- Pasta com os pacotes do infra para cada versão do delphi.
Infra\Packages\D7\Build 	- Pasta para os arquivos compilados do infra na versão 7 do Delphi.
Infra\Core\			- Pasta que mantem o código fonte do Infra.
	Aspect			- classes do InfraAspect, programação orientada a aspectos.
	Common			- classes básicas e servicos principais dos frameworks.
	GUIBuilder		- classes básicas do gerador automático de tela.
	LayoutManager		- classes gerenciador de layouts de tela.
	MVP			- classes básicas para o Model-View-Presenter (a ser refatorado).
	Persistence		- classes do InfraHibernate, persistencia de dados.
	Reflect			- classes do InfraReflect, reflexao do modelo.
	Templates		- arquivos template para listas.
	Valuetype		- classes do InfraType, para substituição dos tipos nativos.
Infra\Demos\			- Não usar estes demos, veja fontes dos testes unitários.
Infra\Documents\		- Pasta que mantem toda a documentação e diagramas.
Infra\Third\			- Pasta com links e tutorial para instalação de bibliotecas 
				  de terceiros (alguns de uso opicional, e configurado no Infra.Inc).
Infra\UnitTests\		- Pasta com testes unitários de cada framework.
